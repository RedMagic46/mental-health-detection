# MindCare — Mental Health Detection

Platform deteksi dini kesehatan mental berbasis web yang menggunakan Next.js dan Supabase. Proyek ini mencakup fitur kuesioner skrining rule-based, sistem autentikasi aman, riwayat penilaian, serta manajemen konsultasi melalui dashboard admin.

## Tech Stack
- **Frontend/Backend**: Next.js 16 (App Router)
- **Database**: Supabase (PostgreSQL)
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Security**: JWT & HttpOnly Cookies
